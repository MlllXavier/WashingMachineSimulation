# WashingMachineSimulation
软件工程课程实践作业，通过模拟洗衣机的运行过程来学习软件的逻辑处理。

经过现场调研，展开对系统的功能需求分析，利用面向对象的思想分析问题，然后对用例进行描述，分析出类图、状态图、时序图、活动图等。
详细UML图见洗衣机模拟系统实践报告。
